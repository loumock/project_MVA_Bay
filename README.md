# project_MVA_Bay
Project in Baysian Machine Learning on the subject of baysian additive regression trees

This project was realized in the master MVA.

This project contains a report and a juperter file.

This project was on the theme of bayesian additive regression trees and is based on the results of the following paper : https://arxiv.org/pdf/0806.3286

The aim of this project was to understand a the theoretical, computational and/or empirical methods developed in the paper, to emphasize the strengths and weaknesses of the paper, and to apply it to real data of our choice. The whole point is to read the paper with a critical mind.
